npm run build --> bundles the app into static files for produvtion
npm test --> starts the test runner
npm run eject --> removes this tool and copies build dependencies, configuration files and scripts into the app directory. If yo udo this, you cant co back!!!
npx create-react-app tiedostonimi --> luo uusi projekti
npm i --> jos haet esim. gitistä react -projektin, tällä komennolla haetaan nodet projektiin (nehän eivät siirry gitiin eivätkä sieltä tule myöskään vakiona)
